# Performance

This website will be deployed publicly.

Performance is critical.

Requirements:

* Lighthouse score above 90
* Fast First Contentful Paint
* Fast Largest Contentful Paint
* Small JavaScript bundle
* Lazy load images
* Lazy load heavy components if needed
* Compress images
* Prefer SVGs
* Minimize layout shifts
* Avoid unnecessary animation libraries
* Keep memory usage low

Always prioritize performance over visual complexity.

---

# Mobile Responsiveness

This website **must be mobile friendly**.

It should work beautifully on:

* Mobile phones
* Tablets
* Laptops
* Desktop monitors
* Ultrawide displays

Do not simply shrink the desktop version.

Create layouts specifically optimized for smaller screens.

Touch interactions should feel natural.

Buttons should be easy to tap.

Navigation should work smoothly on mobile.

The Minecraft aesthetic should remain intact across all screen sizes.

---
